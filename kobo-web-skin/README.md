# Kobo Web Skin

This folder will hold the CARE-AI / Child-Care Thrive web skin requirements and implementation notes for the self-hosted KoboToolbox web platform.

## Intended scope

- Login page branding
- Header/logo replacement
- Favicon replacement
- Colour/token overrides
- Footer/help/support copy
- Light terminology guidance

## Guardrail

Do not change Kobo core submission, XLSForm, authentication, sync, export, or API logic unless explicitly approved.
