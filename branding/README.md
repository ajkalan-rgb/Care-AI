# Branding

This folder is the source of truth for CARE-AI / Child-Care Thrive branding.

## Folder use

- `source/` stores original uploaded brand files exactly as received.
- `processed/` stores cleaned, cropped, resized, transparent, icon, splash, and web-ready versions.

## Upload originals here

Please upload:

- `branding/source/hspn-ribbon-logo-original.png`
- `branding/source/child-care-thrive-splash-original.png`

Do not overwrite originals after upload. Any edits should be saved into `branding/processed/`.
