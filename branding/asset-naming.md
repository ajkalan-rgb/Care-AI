# Asset Naming

The original uploaded brand files may keep their original names in `branding/source/`.

For implementation work, create clean processed versions using these canonical names in `branding/processed/`:

| Canonical file | Purpose |
|---|---|
| `hspn-ribbon-logo-horizontal.png` | Web header / powered-by branding |
| `hspn-ribbon-logo-transparent.png` | Clean transparent HSPN logo for flexible use |
| `child-care-thrive-splash-portrait.png` | Android splash screen source |
| `child-care-thrive-app-icon.png` | Square source for Android launcher icon |
| `child-care-thrive-favicon.png` | Web favicon source |
| `care-ai-login-logo.png` | Kobo web login page logo |
| `care-ai-header-logo.png` | Kobo web header logo |

## Current source assets

The source assets in `branding/source/` are the raw uploaded originals. Do not edit them directly. If the original filenames are not standardised, that is acceptable; the processed outputs above are the files that must use controlled names.
