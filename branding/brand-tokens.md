# CARE-AI / Child-Care Thrive Brand Tokens

## Naming

- Platform: CARE-AI
- Programme / field app: Child-Care Thrive
- Partner / sponsor: HIV Survivors & Partners Network
- Tagline: Data Collection for Better Child Health

## Core colours

| Token | Hex | Use |
|---|---:|---|
| Primary blue | `#006FE6` | Child-Care Thrive primary action colour |
| Deep navy | `#001F5B` | Headings, app bars, serious clinical UI elements |
| Ribbon red | `#E60000` | HSPN ribbon/logo accent only |
| Soft blue background | `#EAF4FF` | Light panels and background shapes |
| Clinical grey | `#6B7280` | Secondary text |
| White | `#FFFFFF` | Backgrounds and cards |

## Asset rules

- Keep raw uploaded images in `branding/source/`.
- Store cleaned/resized assets in `branding/processed/`.
- Do not use the full text-heavy HSPN logo as the Android launcher icon.
- Use the Child-Care Thrive identity for the fieldworker-facing app.
- Use HSPN as a powered-by/co-branding mark.
